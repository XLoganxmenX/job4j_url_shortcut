# * job4j_url_shortcut*

job4j_url_shortcut - RESTful API сервис создания коротких ссылок. Релизован на языке Java с использованием фреймворка
Spring Boot + Security. Сервис предоставляет функционал для регистрации сайтов, авторизации пользователей, преобразования URL и сбора статистики.

### Основной функционал приложения:
1. **Регистрация сайта**.

   Сервис позволяет различным сайтам регистрироваться в системе. При регистрации каждому сайту выдается уникальная пара логина и пароля.
2. **Авторизация**.

   Авторизация осуществляется через JWT (JSON Web Token). Пользователь отправляет POST запрос с логином и паролем для получения токена.
3. **Регистрация URL**.

   После регистрации сайта пользователь может отправлять ссылки на преобразование. Сервис возвращает уникальный код, ассоциированный с оригинальным URL.
4. **Переадресация**.

   Переадресация осуществляется без авторизации. Когда сайт отправляет ссылку с кодом, сервис возвращает ассоциированный адрес с HTTP статусом 302.
5. **Статистика**.

   Сервис собирает статистику по количеству вызовов каждого адреса.

**Стек проекта:**
- **Язык**: Java 17
- **Сборка проекта**: Maven 3.9.6
- **Фреймворк**: Spring Boot 2.7.13
- **База данных**: PostgreSQL 16.2
- **Миграции БД**: Liquibase 3.8
- **ORM**: Hibernate 42.7.3

**Запуск проекта:**

- Установить требуемое в проекте ПО (JDK, PostgreSQL);
- Создать базу shortcut и параметрами подключения из hibernate.cfg.xml;
- Скачать и распаковать проект;
- Через терминал перейти в папку проекта и запустить его двумя командами : "mvn clean install", затем "mvn spring-boot:run";

**Запуск проекта через Docker:**

- Сборка: docker-compose build
- Запуск: docker-compose up -d
